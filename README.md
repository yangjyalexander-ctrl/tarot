# ARCANA · 星月塔罗

> 以星辰为引 · 以月光为镜

一个庄重神秘的塔罗牌在线占卜程序，采用电影级视觉设计，融合古典神秘学与现代网页技术。

## 功能

- **三种牌阵**：单牌占卜 / 时间之流（过去现在未来）/ 凯尔特十字（十牌全解）
- **22张大阿卡纳牌**：完整牌义解读，含正位与逆位
- **流星星空背景**：动态粒子与流星效果
- **氛围音乐**：Web Audio API 生成的神秘ambient音效
- **综合运势分析**：感情、事业、财运、整体运势评分

## 设计

- 电影级神秘主义美学
- 金色粒子与流星动画
- 庄重神秘的视觉风格
- 响应式设计，支持移动端
